# WKVCDeallocMonitor

辅助开发工具，用于实时监听ViewController的释放情况，当出现viewcontroller为释放时，给予leak提示。

## 要求

Xcode6.0+, iOS6.0+

## 安装方式

* `WKVCDeallocMonitor ` 支持使用 `CocoaPods` 引入, 在 `podfile` 文件中添加以下代码:

    ```ruby
    pod 'WKVCDeallocMonitor'
    ```
    
## 使用方式

	无需引入任何文件，pod成功后即可使用
